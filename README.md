# PontoComum

PontoComum é uma plataforma web destinada à divulgação e descoberta de eventos comunitários.  
O projeto foi desenvolvido como parte da disciplina PI: Desenvolvimento de Sistemas Web,
utilizando HTML, CSS, JavaScript e LocalStorage, além de integração com API de geocodificação e mapa.


Funcionalidades:

- Cadastro de eventos com geocodificação automática (API Nominatim – OpenStreetMap)
- Exibição de eventos cadastrados em lista
- Página de detalhes com mapa interativo (Leaflet.js)
- Cadastro e login de usuários utilizando LocalStorage
- Controle de autenticação com SessionStorage
- Interface responsiva
- Armazenamento local dos dados (sem backend)

Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **LocalStorage / SessionStorage**
- **Leaflet.js** (renderização do mapa)
- **Nominatim API** (para obter coordenadas)

 Instalação

Este projeto não requer instalação de dependências.

1) Baixe o Repositório: git clone https://github.com/barddz/PontoComum.git
2) Acesse a pasta do projeto
3) Abra o arquivo index.html no navegador, clicando duas vezes no arquivo index.html, ou usando LiveServer no VScode
